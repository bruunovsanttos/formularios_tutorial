# formularios_tutorial
Repositório de tutorial de criação de formulários

O tutorial utlizado foi o da Rafaella Ballerine [Formulários HTML e CSS](https://www.youtube.com/watch?v=wwqOJ2o84S4)

Nesse tutorial é explorado a criação de checkbox, radiobox, campos que são obrigatórios para preenchimento, e estilização de como ficar o formulário com o css criado por ela.
